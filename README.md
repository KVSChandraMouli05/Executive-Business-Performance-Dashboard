# Executive-Business-Performance-Dashboard
## Overview
This project presents an executive-level Power BI dashboard analyzing business performance using retail transactional data.
The dashboard focuses on revenue, profitability, growth trends, and operational efficiency.

## Key Metrics
- Total Revenue
- Total Profit
- Profit Margin %
- Average Order Value
- Total Orders

## Key Analyses
- Monthly sales trend analysis
- Category-wise sales and profit comparison
- Region-wise performance analysis
- Interactive date filtering

## Tools Used
- Power BI
- DAX
- Retail transactional dataset (Global Superstore)

## Files
- `.pbix` file contains the interactive dashboard
- PDF and images are provided for quick preview
